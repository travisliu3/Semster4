WEB422 Week 1 Demo - WEB Services/APIs

Example Features:

  • Node.js and Express.js
  • No database is used

Items (WEB API Template CRUD operations for an entity - item)

  • R - Get all items /api/items
    • Get all with filters /api/items?page=2
  • R - Get one item /api/items/15
  • C - Create new item - test using Thunder Client or Postman
  • U - Update existing item - test using Thunder Client or Postman
  • D - Delete item - test using Thunder Client or Postman

Colours Entity

  • Get all colours /api/colors
  • Get one colour /api/colors/2

Employees Entity

  • Get all employees /api/employees
    • Get all with filters /api/employees?page=5
  • Get one employee /api/employees/75
  • Update existing employee. This example allow that the json object in req body does not have to include the "employeeId" property.
